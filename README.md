# Footwear Website

This is a responsive Footwear Website developed using HTML and CSS. The website showcases different types of footwear, including sports shoes, sneakers, formal shoes, boots, sandals, and more. Users can browse product categories, view trending collections, check upcoming sales, and explore available payment methods through a clean and user-friendly interface.

## Features
- Responsive Navigation Bar
- Footwear Categories Section
- Trending Products Display
- Upcoming Sales Information
- Payment Methods Section
- Contact Information
- Mobile-Friendly Design

## Technologies Used
- HTML5
- CSS3
